# vue-i18n集成与构建优化

## 集成vue-i18n基础功能，初步探索

安装vue-i18n依赖包，并按照官网进行配置

在main.ts+index.vue中编写基础代码，测试功能

使用 composition API 风格引入$t，设置legacy: false，避免全局引入

添加语种切换功能，参考官网advanced/composition#locale-changing



## 集成unplugin-vue-18n，实现语种数据懒加载

### 创建locales文件夹

在根目录创建locales文件夹，存放各语种的messages

创建zh-CN和en两种语言数据吗，格式为js

### 安装unplugin-vue-18n

安装unplugin-vue-18n。该插件会自动解析目标路径下的yml或json文件，生成js数据对象

按照官网进行vite相关配置，语种文件夹为locales

### 实现语种数据的懒加载

按照vue-i18n官网的Lazy Loading指示，实现messages的懒加载

在src/modules下创建i18n.ts，拷贝官方懒加载代码，准备以插件方式实现国际化

参考vitesse库的i18n.ts，优化懒加载方式

### 以插件方式使用vue-i18n

为i18n.ts添加install方法

在main.ts中，为vue实例添加i18n.ts插件

### 调整vue-i18n的使用方式

调整main.ts+index.vue中的基础代码，更新i18n的使用方式

### 集成vsc插件i18n Ally

安装vsc插件i18n Ally，使代码中自动显示目标翻译结果

调整i18n Ally的配置项，使功能正常执行
